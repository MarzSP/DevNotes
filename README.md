# DevNotes
DevNotes: documenting the journey from 'hello world' to high-level systems. 

Visit the blog at: https://marzsp.github.io/DevNotes/

## Features:

Written in Markdown (.md/.mdx)(/src/content/blog)

Built with Astro 

Custom styling with CSS variables

Filterable post list and date formatting


The site is automatically built and deployed through GitHub Actions to GitHub Pages every time you push to the main branch.
